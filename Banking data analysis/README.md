# Исследование надежности заемщиков

## Задача
На основе статистики о платёжеспособности клиентов исследовать влияет ли семейное положение и количество детей клиента на факт возврата кредита в срок

## Навыки и инструменты
- python
- pandas
- предобработка данных

## Вывод
На основе данных кредитного отдела банка проведено исследование влияния семейного положения и количества детей на факт погашения кредита в срок. Была получена информация о данных. Определены и обработаны пропуски. Заменены типы данных на соответствующие хранящимся данным. Удалены дубликаты. Категоризованы данные.
